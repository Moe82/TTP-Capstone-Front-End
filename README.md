## Getting Started

To get a local copy up and running follow these simple example steps.

<!-- INSTALLATION -->

### Installation

1. Clone the repo

   ```sh
   git clone https://github.com/Moe82/TTP-Capstone-Front-End.git
   ```

2. Install NPM packages

   ```sh
   npm install
   ```

3. Start your app up by running

   ```sh
   npm start
   ```
